# De Patria a Patria

## Descripción del Proyecto

Este proyecto consiste en un **sitio web responsivo de una galería de arte**, titulado _"De Patria a Patria"_.  
El sitio presenta sobre distintas culturas y ciudades, mostrando historias, fotografías y reflexiones que conectan a personas de diferentes partes del mundo que trabajan en TRIPLETEN.

## Funcionalidad

- Presentación de una portada con título, subtítulo e imagen principal.
- Sección introductoria con una cita inspiradora y una explicación del propósito de la galería.
- Diseño visual limpio y adaptable a diferentes dispositivos (responsivo).
- Enfoque en la experiencia del usuario con imágenes y textos que invitan a la reflexión.

## Tecnologías y Técnicas Utilizadas

- **HTML5** → estructura semántica del sitio.
- **CSS3** → estilos y diseño responsivo.
- **Metodología BEM** para la organización y mantenibilidad de las clases CSS.
- **Buenas prácticas de accesibilidad** (uso de `alt`, etiquetas semánticas y metadatos).
